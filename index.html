40 Most Common Unix Commands

Here is a list of the most commonly used Unix commands for system administration, file manipulation, and more:

1. ls      - List directory contents
2. cd      - Change directory
3. pwd     - Print working directory
4. touch   - Create an empty file
5. cat     - Display file contents
6. cp      - Copy files or directories
7. mv      - Move/rename files or directories
8. rm      - Remove files or directories
9. mkdir   - Create a directory
10. rmdir   - Remove an empty directory
11. chmod   - Change file permissions
12. chown   - Change file owner or group
13. find    - Search for files or directories
14. grep    - Search for patterns in files
15. echo    - Display a string
16. man     - Display command manual
17. head    - View the first lines of a file
18. tail    - View the last lines of a file
19. wc      - Count lines, words, and characters in a file
20. sort    - Sort lines in a file
21. uniq    - Remove duplicate lines
22. diff    - Compare two files line by line
23. tar     - Archive files
24. zip     - Compress files
25. unzip   - Extract compressed files
26. ps      - Display active processes
27. kill    - Terminate processes by PID
28. top     - Show real-time processes
29. df      - Display disk space usage
30. du      - Show directory space usage
31. uname   - Display system information
32. whoami  - Show current user
33. date    - Display or set the system date
34. cal     - Display a calendar
35. ping    - Check network connectivity
36. wget    - Download files from the web
37. curl    - Transfer data from URLs
38. ssh     - Remote login to another machine
39. scp     - Copy files between systems
40. exit    - Log out or exit the shell

Introduction to Python

Python is a versatile programming language widely used in web development, data analysis, artificial intelligence, and scientific computing. It is known for its simplicity, readability, and a large standard library, making it an ideal choice for beginners and professionals alike.

Why Learn Python?

  • Python has a simple syntax, making it easy to learn for beginners.
  • It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.
  • It has a vast collection of libraries and frameworks, such as NumPy, Pandas, TensorFlow, and Django.
  • Python is widely used in industries like web development, data science, AI, machine learning, and more.

Subtopic: Variables and Data Types

In Python, variables are used to store data, and they donâ€™t require explicit declaration. Data types are automatically assigned based on the value.

  • String: A sequence of characters enclosed in single (' ') or double (" ") quotes. Example: 
    name = "John Doe"
  • Integer: A whole number without a decimal point. Example: 
    age = 25
  • Float: A number with a decimal point. Example: 
    height = 5.9
  • List: An ordered collection of items, which can be of mixed types. Lists are mutable. Example: 
    fruits = ["apple", "banana", "cherry"]
  • Tuple: An ordered collection of items, similar to a list, but immutable. Example: 
    coordinates = (10, 20, 30)
  • Dictionary: A collection of key-value pairs. Example: 
    person = {"name": "Alice", "age": 30, "city": "New York"}
  • Boolean: Represents either True or False. Example: 
    is_active = True

Subtopic: Control Flow Statements

Control flow statements allow you to control the execution of code based on conditions or loops.

  • If-Else: Used to execute code conditionally. Example: 
    if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")
  • For Loop: Used to iterate over a sequence (e.g., list, tuple, or string). Example: 
    for fruit in fruits:
    print(fruit)
  • While Loop: Used to execute a block of code repeatedly as long as a condition is true. Example: 
    count = 0
while count < 5:
    print(count)
    count += 1

Subtopic: Functions

Functions are blocks of reusable code that perform a specific task. They are defined using the def keyword.

# Example: A function to calculate the square of a number
def square(num):
    return num * num

result = square(5)
print(result)  # Output: 25

Subtopic: Modules and Libraries

Python comes with a rich set of built-in libraries and allows you to use external libraries by importing them.

  • Built-in Modules: Examples include math, datetime, and os. Example: 
    import math
print(math.sqrt(16))  # Output: 4.0
  • External Libraries: These need to be installed using a package manager like pip. Examples:
    • NumPy: Used for numerical computations.
    • Pandas: Used for data manipulation and analysis.
    • Matplotlib: Used for data visualization.
